    📖 How to Play

    🎯 The Goal

    Mine blocks → earn coins → buy upgrades → mine faster → climb the Hall of Fame.

    ⛏ Basics

    • Click a block to hit it with your hammer. Each hit deals Drill Power damage.
    • Break a block to earn coins. Rarer ores = more coins.
    • Ores: Coal (common) · Iron · Gold · Diamond · Alien Crystal (rarest, biggest reward).
    • Blocks respawn shortly after breaking.

    🛠 Upgrades (left panel / ☰ button on mobile)

    • Drill Power — more damage per click.
    • Crit Chance — chance for ×5 damage on a click.
    • Auto-Drill — robots that mine for you every second, even offline (up to 8 h).
    • Coin Multiplier — bigger reward for every break.

    🔘 Top Bar Buttons

    • ☰ Upgrades — opens the upgrades drawer on mobile.
    • Hall of Fame — leaderboard. Pay 5,000 coins to enter your name + a profile link. Spend more coins to boost your score.
    • ☁ Save to cloud — uploads your save to the cloud. Your save file is your account — keep it safe.
    • Export — downloads a .json backup. Includes your cloud ID.
    • Import — load a previously exported save (e.g. on a new device).
    • Reset — wipes local data and starts fresh with a new cloud ID. Irreversible.
    • 👤 Account & Backup — link this save to an email and get a one-time backup code. On a new device, enter your email + code to restore your progress. Lose the code = lose access — save it somewhere safe.
    • 📖 Docs — this guide.
    • Developer — about the creator.

    💾 Saving & Cloud

    The game auto-saves to your browser. Click ☁ Save to cloud to back up online — no login needed for the save itself.

    To recover your save on a different device, set up a backup code via the 👤 Account & Backup button: pick an email, we generate a 12-character code, you save it. Then on the new device: enter email + code → save restored. You can rotate the code anytime if you think it leaked.

    🏆 Hall of Fame Tips

    Joining costs 5,000 coins. The first boost costs 10,000, the next 20,000, and so on — each boost doubles. Display names must be unique. Add an Instagram / LinkedIn / X URL so people can find you.

    📱 Mobile

    Tap blocks to mine. Use the ☰ Upgrades button to open the upgrades drawer. Long-pressing is disabled to avoid accidental selection.
